# Tax Refund System

### Introduction
This project aims to simulate the departure tax refund process and provide a self-service method for foreign tourists in mainland China.

### Main Features
+ The <b><u>merchant</u></b> create invoices and application forms;
+ The <b><u>customs</u></b> review the application form;
+ The <b><u>agency</u></b> refunds taxes to tourists;
+ A <b><u>management system</u></b> for managing user accounts and visualizing data.

### Technologies (languages & frameworks)
+ Backend: SpringBoot/Maven/MyBatis;
+ Database: MySQL;
+ Frontend: HTML/CSS/JavaScript/jQuery/Echarts.
